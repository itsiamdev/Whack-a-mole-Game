# 🐭 Whack-a-Mole Game 🕳️

A fun and interactive Whack-a-Mole game built with HTML, CSS, and JavaScript. ⏰

## 🎮 How to Play

1. Click the "Start" button to begin the game. ▶️
2. Moles will randomly appear in the holes. 🐭
3. Click on the moles as quickly as possible to whack them and increase your score. 👆💥
4. The game lasts for 10 seconds. ⏱️
5. After the time is up, your final score is displayed, and you can click "Try again?" to play again. 🔄

## ✨ Features

- Random mole appearances with varying durations. 🎲
- Score tracking. 📊
- Simple and responsive design. 📱
- Prevents cheating by checking for trusted events. 🛡️

## 📁 Files

- `index.html`: The main HTML structure. 🌐
- `style.css`: Styling for the game layout and animations. 🎨
- `script.js`: Game logic and event handling. ⚙️

## 🚀 Running the Game

Open `index.html` in a web browser to start playing. 🌍
